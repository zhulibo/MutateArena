修改窗口模式和分辨率后，不知道为什么退出游戏时会恢复默认
如果想永久修改，请手动在此文件（拥有最高优先级）覆盖配置项 C:\Users\[你的电脑名字]\AppData\Local\MutateArena\Saved\Config\Windows\GameUserSettings.ini
可配置项参考 https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/Engine/UGameUserSettings

After changing the window mode and resolution, for some reason, the game reverts to the default settings upon exiting.
To permanently change the settings, please manually overwrite the configuration options in this file (which has the highest priority): C:\Users\[Your Computer Name]\AppData\Local\MutateArena\Saved\Config\Windows\GameUserSettings.ini.
Refer to https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/Engine/UGameUserSettings for configurable options.
